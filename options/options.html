Open this page once in a normal tab first so the browser solves Anubis and stores the clearance cookie. The poller then reuses that session.

Publish to GitHub optional

Leave disabled for local-only use. Enable to push data.json to a repo so a GitHub Pages dashboard can serve it. Use a fine-grained PAT scoped to just this repo with Contents: read & write. The token is stored only in this browser's local extension storage.

Debug

The extension logs each poll and keeps the last raw SoL response plus the parse result. Export the bundle and share it to diagnose parsing or fetch issues. The GitHub token is automatically redacted.
